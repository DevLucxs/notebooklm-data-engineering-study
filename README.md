# Mini Guia de Estudos com NotebookLM – Engenharia de Dados

Este projeto foi desenvolvido como parte de um desafio de aprendizado na DIO utilizando o NotebookLM.

O objetivo da atividade é explorar o uso de Inteligência Artificial como ferramenta de estudo, reunindo fontes abertas, organizando perguntas estratégicas e gerando resumos estruturados a partir do material analisado.

Embora o desafio original sugerisse temas de educação financeira, optou-se por aplicar a metodologia em um tema técnico de interesse profissional: Engenharia de Dados.

A proposta consistiu em:

* Selecionar fontes confiáveis sobre engenharia de dados
* Criar um caderno temático no NotebookLM
* Elaborar perguntas estratégicas para explorar o material
* Gerar resumos estruturados e um glossário de conceitos
* Criar prompts reutilizáveis para futuras revisões de estudo

O resultado é um mini guia de estudo que pode ser utilizado como base para aprendizado progressivo na área de engenharia de dados.

## Ferramentas utilizadas

* NotebookLM
* Documentação técnica de ferramentas de dados
* Livros e artigos sobre engenharia de dados

## Estrutura do projeto

* `fontes/` → lista de materiais utilizados
* `prompts/` → prompts testados no NotebookLM
* `guia/` → resumo estruturado do conteúdo estudado
